# Voice Notes

Это современное веб-приложение для создания заметок с поддержкой голосового ввода. Проект ориентирован на быстрые записи, удобную навигацию и минималистичный интерфейс.

**Демо: https://myvoicenotes.vercel.app**

## Возможности
- Создание, просмотр, редактирование и удаление заметок
- Импорт и экспорт заметок
- Возможность поставить на заметку пароль
- Запись текста голосом (непрерывное прослушивание) или вручную
- Поиск по названию и содержимому заметок
- Сортировка: по названию, по дате создания, по дате изменения
- Выбор темы оформления: светлая (по умолчанию) и темная тема
- Выбор языка: русский (по умолчанию) и английский
- Адаптивный интерфейс

## Установка и запуск

```bash
git clone https://github.com/wwyynnee/VoiceNotes.git
cd VoiceNotes
npm install
npm start
```
